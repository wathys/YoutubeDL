# YoutubeDL
Téléchargeur YouTube

Pour utiliser l'application, lancez `YoutubeDL`.

Applications requises (voir `requirements.txt`) :
- yt-dlp
- ffmpeg-python
- requests

Pour installer automatiquement les dépendances, exécutez `install_requirment.py`.

Plus d'informations : https://github.com/wathys
