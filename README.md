# YoutubeDL
Script pour télécharger des vidéo youtube en haute qualité.

Pour utiliser le script, lancez `YoutubeDL.pyw`.

logiciels requises (voir `requirements.txt`) :
- yt-dlp
- ffmpeg-python
- requests

Pour installer automatiquement les dépendances, exécutez `install_requirment.py`.

Plus d'informations : https://github.com/wathys
